# Query

## 1. 什么是 Query

Query 是一个查询股票指标数据的工具，它有很多查询参数，可以根据自己的需求来选择合适的参数，比如可以查询某只股票的市值、市盈率等指标，也可以查询技术指标 SMA EMA 等。

同时，Query 还支持保存查询条件，方便下次使用。也可以分享你的 Query 给你的朋友。


## 2. 如何使用 Query

打开 [BreadthApp Query](https://breadth.app/query) 页面，你可以看到一个 Query 的页面，如下图所示：

![Query](/query/query.png)

### 2.1 基本搜索

你可以在输入框输入股票代码，然后点击搜索按钮，即可查询到该股票的指标数据。

![Query](/query/search-by-symbol.png)


### 2.2 高级搜索

高级搜索支持 `Filter` 和 `Sort` 两种条件；通过界面中 `Add Filter` 唤出条件面板。

- Filter: 搜索条件，可以根据市值、市盈率、技术指标等来筛选股票。
- Sort: 默认降序排序，可以根据某个指标来排序，比如按照市值从大到小排序。

>点击 `Add Filter` 按钮，即可显示 Filter 面板。

![Filter panel](/query/filter-panel.png)

>添加需要的条件到 Filter 中，比如添加市值、涨跌幅等条件。

![Search Query](/query/filter-sort-panel.png)

>点击 `Search` 按钮，即可查询到符合条件的股票.

![Query Result](/query/query-result.png)


## 3. 保存查询

合适的 Query 通过点击 `Save` 按钮，输入 Query 名称，即可保存 Query。

![Click Save](/query/click-save.png)

![Save Query](/query/create-query.png)

## 3.1 查看已保存的 Query

点击 `My Query` 按钮，即可查看已保存的 Query。

>点击 `My Query` 按钮，即可查看已保存的 Query。

![Click My Query](/query/click-my-query.png)

![My Query](/query/saved-query.png)

## 3.2 使用已保存的 Query

点击 `+` 图标，已保存的 Query 会出现在条件区域。点击 `Search` 按钮搜索。

![Use Query](/query/use-query.png)

如果条件区域已有查询条件，点击 + 图标会覆盖原有查询条件。会弹出提示框，确认后即可使用已保存的 Query。

![Use Query](/query/use-query-waring.png)

## 4. 分享 Query

分享 Query 之后任何知道链接的人都可以查看你的 Query。

![Click Share](/query/click-share-query.png)

确认后，会生成一个链接，复制到粘贴板，即可分享给你的朋友。 同时 Query 的状态会变成 `PUBLIC`。

![Share Alert](/query/share-alert.png)

![Share Query](/query/query-public.png)

比如这个演示生成的 Query: https://breadth.app/query/PjCYa8gdQYJeUiwkx3ZbKg

### 4.1 取消分享 Query

再次点击 `Share` 按钮，出现提示框，点击按钮，即可取消分享 Query。状态会变成 `PRIVATE`。

![Unshare Query](/query/private-query.png)

## 5. 删除 Query

点击 `Delete` 按钮，即可删除 Query。删除后你将无法查看 Query；已经分享的 Query 也将无法访问。

![Delete Query](/query/delete-query.png)
