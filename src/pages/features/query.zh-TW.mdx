# Query

## 1. 什麽是 Query

Query 是一個查詢股票指標數據的工具，它有很多查詢參數，可以根據自己的需求來選擇合適的參數，比如可以查詢某只股票的市值、市盈率等指標，也可以查詢技術指標 SMA EMA 等。

同時，Query 還支持保存查詢條件，方便下次使用。也可以分享你的 Query 給你的朋友。


## 2. 如何使用 Query

打開 [BreadthApp Query](https://breadth.app/query) 頁面，你可以看到一個 Query 的頁面，如下圖所示：

![Query](/query/query.png)

### 2.1 基本搜索

你可以在輸入框輸入股票代碼，然後點擊搜索按鈕，即可查詢到該股票的指標數據。

![Query](/query/search-by-symbol.png)


### 2.2 高級搜索

高級搜索支持 `Filter` 和 `Sort` 兩種條件；通過界面中 `Add Filter` 喚出條件面板。

- Filter: 搜索條件，可以根據市值、市盈率、技術指標等來篩選股票。
- Sort: 默認降序排序，可以根據某個指標來排序，比如按照市值從大到小排序。

>點擊 `Add Filter` 按鈕，即可顯示 Filter 面板。

![Filter panel](/query/filter-panel.png)

>添加需要的條件到 Filter 中，比如添加市值、漲跌幅等條件。

![Search Query](/query/filter-sort-panel.png)

>點擊 `Search` 按鈕，即可查詢到符合條件的股票.

![Query Result](/query/query-result.png)


## 3. 保存查詢

合適的 Query 通過點擊 `Save` 按鈕，輸入 Query 名稱，即可保存 Query。

![Click Save](/query/click-save.png)

![Save Query](/query/create-query.png)

## 3.1 查看已保存的 Query

點擊 `My Query` 按鈕，即可查看已保存的 Query。

>點擊 `My Query` 按鈕，即可查看已保存的 Query。

![Click My Query](/query/click-my-query.png)

![My Query](/query/saved-query.png)

## 3.2 使用已保存的 Query

點擊 `+` 圖標，已保存的 Query 會出現在條件區域。點擊 `Search` 按鈕搜索。

![Use Query](/query/use-query.png)

如果條件區域已有查詢條件，點擊 + 圖標會覆蓋原有查詢條件。會彈出提示框，確認後即可使用已保存的 Query。

![Use Query](/query/use-query-waring.png)

## 4. 分享 Query

分享 Query 之後任何知道鏈接的人都可以查看你的 Query。

![Click Share](/query/click-share-query.png)

確認後，會生成一個鏈接，復製到粘貼板，即可分享給你的朋友。 同時 Query 的狀態會變成 `PUBLIC`。

![Share Alert](/query/share-alert.png)

![Share Query](/query/query-public.png)

比如這個演示生成的 Query: https://breadth.app/query/PjCYa8gdQYJeUiwkx3ZbKg

### 4.1 取消分享 Query

再次點擊 `Share` 按鈕，出現提示框，點擊按鈕，即可取消分享 Query。狀態會變成 `PRIVATE`。

![Unshare Query](/query/private-query.png)

## 5. 刪除 Query

點擊 `Delete` 按鈕，即可刪除 Query。刪除後你將無法查看 Query；已經分享的 Query 也將無法訪問。

![Delete Query](/query/delete-query.png)
